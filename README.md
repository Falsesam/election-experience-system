# 🗳️ Election Experience System

## 🚀 Overview

An interactive system designed to help users understand the election process through guided interaction, simulation, and adaptive responses.

Built as part of **Virtual Prompt Wars** by Google for Developers.

---

## 💡 Features

* 💬 Assistant Mode (query-based responses)
* 🧭 Guide Mode (step-by-step explanation)
* 🎮 Simulation Mode (decision-based scenarios)
* 🎯 Mission Mode (guided interaction tasks)
* 🧠 Adaptive behavior (basic user interaction tracking)
* 🌐 Streamlit web interface

---

## 🧠 Architecture

User Input → Validator → Engine → Knowledge Base → Response

## ▶️ Run Locally

### Install dependencies

```bash
python -m pip install -r requirements.txt
```

### Run CLI

```bash
python main.py
```

### Run Web App

```bash
python -m streamlit run app.py
```
